# Mnemonic Spire

A small TypeScript canvas game project scaffolded for iterative web game development.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Development Notes

- Main game loop: `src/main.ts`
- Global test hooks:
  - `window.render_game_to_text()`
  - `window.advanceTime(ms)`
- Fullscreen toggle: `f`
- Movement: arrow keys or WASD
