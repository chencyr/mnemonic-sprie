# Mnemonic Spire

A Phaser 3 + TypeScript + Vite game project scaffolded for iterative web game development.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Development Notes

- Engine: Phaser 3
- App entry: `src/main.ts`
- Global test hooks:
  - `window.render_game_to_text()`
  - `window.advanceTime(ms)`
- Fullscreen toggle: `f`
- Movement: arrow keys or WASD
